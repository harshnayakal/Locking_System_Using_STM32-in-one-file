# Locking_System_Using_STM32-in-one-file
This project is a Two-Factor Authentication (2FA) Locker System developed using the STM32F103C8T6 microcontroller and Embedded C with register-level programming. It combines password and OTP-based authentication using a 4×4 keypad, 16×2 LCD, LEDs, and a buzzer to provide enhanced security with a three-attempt lockout mechanism.
